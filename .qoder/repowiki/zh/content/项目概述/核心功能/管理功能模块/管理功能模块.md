## 项目概述

红灯笼食府管理系统是一个基于 Vue 3 + Node.js + SQLite 的餐饮企业管理解决方案。系统采用前后端分离架构，包含顾客端（C端）和管理端（B端）两大模块。

### 核心特性
- **实时订单管理**：支持实时订单推送和状态变更通知
- **多维度数据统计**：提供详细的经营数据分析
- **灵活的权限控制**：基于角色的访问控制机制
- **数据安全保障**：多重安全防护措施
- **移动端适配**：完整的响应式设计

## 实时推送机制

### SSE 实时推送

系统采用了 Server-Sent Events (SSE) 实现实时推送功能：

```mermaid
sequenceDiagram
participant Client as 管理端客户端
participant SSE as SSE 连接
participant Server as 后端服务器
participant DB as 数据库
Client->>Server : GET /api/admin/events
Server->>SSE : 建立 SSE 连接
SSE-->>Client : connected 事件
SSE-->>Client : 心跳保活 (30秒)
loop 订单状态变更
DB->>Server : 订单状态更新
Server->>SSE : 推送 order_updated 事件
SSE-->>Client : 订单状态更新消息
end
loop 新订单产生
DB->>Server : 新订单创建
Server->>SSE : 推送 new_order 事件
SSE-->>Client : 新订单消息
end
Client->>SSE : 断开连接
SSE->>Server : 清理资源
```
