- **统一入口与中间件编排**：开发环境下，`server/src/dev.ts` 启动 Express 服务器并挂载 Vite 中间件，实现前端热更新与后端 API 代理在同一端口（3000）运行，消除了跨域问题。
- **共享依赖与构建协调**：根目录 `package.json` 统一管理前后端依赖，通过 `tsx` 和 `vue-tsc` 协调 TypeScript 编译；生产环境通过 `scripts/build-production.mjs` 串行执行前端静态资源构建与后端代码编译。
- **容器化交付标准**：`Dockerfile` 与 `docker-compose.yml` 定义了标准化的生产运行时，配置了 SQLite 数据卷持久化、日志轮转及健康检查，确保前后端产物在隔离环境中稳定协同。
