# 修复购物车展开时 `.cart-expanded` 与 `.cart-bar` 之间的边框线条问题

## 摘要

用户反馈：购物车展开后，上方展开区（`.cart-expanded`）与下方确认订单栏（`.cart-bar`）之间出现了一条不应有的分隔线，且左右两侧的边框线条存在 1px 错位。本计划通过将 `outline` 改为 `border` 并按需关闭相邻边的边框，实现两块区域无缝拼接、左右边框完全对齐。

## 当前状态分析

DOM 结构（[HomeView.vue:284-339](file:///c:/Users/赵晨旭/Desktop/红灯笼食府管理系统/src/client/views/HomeView.vue#L284-L339)）：

```
.cart-container
├─ <Transition name="cart-expand">
│   └─ .cart-expanded              ← 展开主体（header + body）
└─ .cart-bar                       ← 确认订单栏（展开时与上方相邻）
```

当前样式问题根源：

1. [HomeView.vue:517-528](file:///c:/Users/赵晨旭/Desktop/红灯笼食府管理系统/src/client/views/HomeView.vue#L517-L528) `.cart-expanded`：
   - `outline: 1px solid var(--color-border-light);`（outline-offset 默认 0，绘制在 border 外侧）
   - `margin-bottom: -1px;`（向上重叠 1px，试图消除中间线）

2. [HomeView.vue:638-643](file:///c:/Users/赵晨旭/Desktop/红灯笼食府管理系统/src/client/views/HomeView.vue#L638-L643) `.cart-expanded + .cart-bar`：
   - `outline: 1px solid var(--color-border-light);`
   - `outline-offset: -1px;`（outline 向内偏移 1px，绘制在 border 内侧）

**问题成因：**

- **中间线条**：两个元素都绘制了 outline，由于 `margin-bottom: -1px` 与 `outline-offset: -1px` 的组合并未让两条 outline 完全重合，中间露出一条可见线条。
- **左右错位**：`.cart-expanded` 的 outline 在 border 外侧（offset 0），`.cart-bar` 的 outline 在 border 内侧（offset -1px），两侧线条横向位置相差 1px，视觉上错位。

**关键约束：** 全局已设置 `box-sizing: border-box`（[src/style.css:117-123](file:///c:/Users/赵晨旭/Desktop/红灯笼食府管理系统/src/style.css#L117-L123)），因此改用 `border` 不会影响元素布局尺寸，方案可行。

## 提议变更

仅修改一个文件：[src/client/views/HomeView.vue](file:///c:/Users/赵晨旭/Desktop/红灯笼食府管理系统/src/client/views/HomeView.vue)

### 变更 1：`.cart-expanded` 样式（第 517-528 行）

**目的**：用 border 替代 outline，并去掉底边框，使下方 `.cart-bar` 能无缝衔接；同时移除不再需要的负 margin。

**修改前：**
```css
.cart-expanded {
  background-color: var(--color-bg-secondary);
  border-radius: var(--radius-xl) var(--radius-xl) 0 0;
  max-height: 40vh;
  display: flex;
  flex-direction: column;
  outline: 1px solid var(--color-border-light);
  overflow: hidden;
  margin-bottom: -1px;
  position: relative;
  z-index: 1;
}
```

**修改后：**
```css
.cart-expanded {
  background-color: var(--color-bg-secondary);
  border-radius: var(--radius-xl) var(--radius-xl) 0 0;
  border: 1px solid var(--color-border-light);
  border-bottom: none;
  max-height: 40vh;
  display: flex;
  flex-direction: column;
  overflow: hidden;
  position: relative;
  z-index: 1;
}
```

变更点：
- 移除 `outline: 1px solid var(--color-border-light);`
- 新增 `border: 1px solid var(--color-border-light);` 与 `border-bottom: none;`
- 移除 `margin-bottom: -1px;`（改用 border 方案后无需重叠消除中间线）

### 变更 2：`.cart-expanded + .cart-bar` 样式（第 638-643 行）

**目的**：用 border 替代 outline，并去掉顶边框，与上方 `.cart-expanded` 的底边缺失处无缝拼接；左右 border 与上方对齐，消除错位。

**修改前：**
```css
.cart-expanded + .cart-bar {
  border-radius: 0 0 var(--radius-xl) var(--radius-xl);
  box-shadow: none;
  outline: 1px solid var(--color-border-light);
  outline-offset: -1px;
}
```

**修改后：**
```css
.cart-expanded + .cart-bar {
  border-radius: 0 0 var(--radius-xl) var(--radius-xl);
  box-shadow: none;
  border: 1px solid var(--color-border-light);
  border-top: none;
}
```

变更点：
- 移除 `outline: 1px solid var(--color-border-light);` 与 `outline-offset: -1px;`
- 新增 `border: 1px solid var(--color-border-light);` 与 `border-top: none;`

## 方案原理

- `.cart-expanded` 显示「上 + 左 + 右」三边边框，底边留空。
- `.cart-bar`（展开态）显示「下 + 左 + 右」三边边框，顶边留空。
- 两者左右 border 均在 `border-box` 内侧、紧贴圆角弧线，位置完全一致 → **消除左右错位**。
- 两者相邻处一方无边框 → **中间无线条**，视觉上形成完整连续的圆角边框。

## 假设与决策

1. **假设**：全局 `box-sizing: border-box` 已生效，改用 border 不会撑大元素，布局保持不变。已通过 [src/style.css:117-123](file:///c:/Users/赵晨旭/Desktop/红灯笼食府管理系统/src/style.css#L117-L123) 验证。
2. **决策**：选择 `border` 而非 `box-shadow` 模拟边框，原因是 border 语义清晰、可单边控制、与现有代码风格一致，且圆角下 border 渲染自然。
3. **决策**：保留 `.cart-expanded` 的 `position: relative; z-index: 1;`，未做改动，避免影响原有层叠上下文。
4. **不改动项**：`.cart-bar`（非展开态）保持原 `box-shadow` 浮起效果；`.cart-expanded-header` 的 `border-bottom` 保持不变（它是 header 与 body 之间的分隔线，与本次问题无关）。
