# 模态窗横向矩形改造计划

## 一、任务概述

将所有基于 `Modal.vue` 组件的模态窗改宽，使其呈现"横长方形"外观。**仅修改宽度，不改动任何内容布局**。自定义模态窗（ConfirmDialog、ClientLoginModal、progress-modal、table-full-modal）不在本次改造范围内。

## 二、当前状态分析

文件：`src/shared/components/Modal.vue`（第 80-93 行）

```css
.modal-sm { width: 100%; max-width: 320px; }
.modal-md { width: 100%; max-width: 480px; }
.modal-lg { width: 100%; max-width: 640px; }
```

当前宽度较窄，模态窗呈现"竖长方形"外观。

## 三、改造方案

**唯一修改文件**：`src/shared/components/Modal.vue`

**修改内容**（第 80-93 行的 max-width 值）：

```css
.modal-sm {
  width: 100%;
  max-width: 480px;  /* 320px → 480px */
}

.modal-md {
  width: 100%;
  max-width: 640px;  /* 480px → 640px */
}

.modal-lg {
  width: 100%;
  max-width: 800px;  /* 640px → 800px */
}
```

**理由**：
- sm 从 320 增至 480px（增 50%）
- md 从 480 增至 640px（增 33%）
- lg 从 640 增至 800px（增 25%）
- 各档保持 160px 等差递增，整体呈现横向矩形外观
- 所有使用 Modal.vue 的模态窗（共 15 处）自动受益，无需逐个修改

## 四、不改动的内容

- **不改动**任何模态窗的内容布局（表单字段排列、信息展示方式等保持原样）
- **不改动**自定义模态窗（ConfirmDialog、ClientLoginModal、progress-modal、table-full-modal）
- **不改动** Modal.vue 的其他样式（header/body/footer 结构、动画等）

## 五、验证步骤

1. 启动开发服务器：`npm run dev`
2. 打开任意使用 Modal.vue 的模态窗，确认宽度明显增加，呈现横向矩形
3. 确认模态窗内容布局未发生变化（表单字段仍为原排列方式）
4. 确认窄屏（<480px）下模态窗仍能正常自适应（因 `width: 100%` + backdrop padding）
