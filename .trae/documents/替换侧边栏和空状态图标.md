# 替换侧边栏和空状态图标

## 任务摘要

根据用户要求，将侧边栏两个导航项的图标替换为 DebugToolsPanel 分组标题中对应的图标，并将 DashboardView 空状态图标替换为侧边栏替换下来的 ClipboardList 图标。

## 当前状态分析

用户选中了 6 个浏览器元素，对应三组替换关系：

| 组别 | 源元素（被替换） | 目标元素（替换为） | 含义 |
|---|---|---|---|
| 1 | button（侧边栏"桌位管理"，ClipboardList） | div.group-header（DebugToolsPanel"桌位管理"，Armchair） | 侧边栏桌位管理图标换成 Armchair |
| 2 | button（侧边栏"库存管理"，Archive） | div.group-header（DebugToolsPanel"库存管理"，Package） | 侧边栏库存管理图标换成 Package |
| 3 | div.empty-icon（DashboardView 空状态，Box） | svg（侧边栏 clipboard-list-icon，ClipboardList） | 空状态图标换成 ClipboardList |

### LayoutView.vue（`src\admin\views\LayoutView.vue`）

- 第 8-19 行 import：引入了 `ClipboardList`（第 11 行）和 `Archive`（第 16 行）
- 第 42 行：`{ icon: ClipboardList, label: '桌位管理', path: '/admin/tables' }`
- 第 44 行：`{ icon: Archive, label: '库存管理', path: '/admin/inventory' }`
- 经 Grep 确认，`ClipboardList` 和 `Archive` 仅在 import 和 navItems 中使用，替换后可从 import 移除

### DashboardView.vue（`src\admin\views\DashboardView.vue`）

- 第 11 行 import：`import { ShoppingBag, DollarSign, Clock, CheckCircle, Eye, XCircle, Trash2, Box, Bell, BellOff, Search } from 'lucide-vue-next'`
- 第 606 行：`<Box :size="48" />`（空状态图标，需替换）
- 第 796 行：`<Box :size="32" />`（另一处使用，**不替换**，因此 `Box` 需保留在 import 中）

## 拟定修改

### 修改 1：LayoutView.vue — 替换侧边栏两个导航项图标

**文件**：`src\admin\views\LayoutView.vue`

1. **import 调整**（第 8-19 行）：
   - 移除 `ClipboardList`（第 11 行）
   - 移除 `Archive`（第 16 行）
   - 新增 `Armchair` 和 `Package`

   修改后 import 块：
   ```ts
   import {
     Home,
     UtensilsCrossed,
     Armchair,
     Settings,
     LogOut,
     Menu,
     X,

     Users,
     Wrench
   } from 'lucide-vue-next'
   ```

2. **navItems 调整**：
   - 第 42 行：`{ icon: ClipboardList, ... }` → `{ icon: Armchair, ... }`
   - 第 44 行：`{ icon: Archive, ... }` → `{ icon: Package, ... }`

### 修改 2：DashboardView.vue — 替换空状态图标

**文件**：`src\admin\views\DashboardView.vue`

1. **import 调整**（第 11 行）：新增 `ClipboardList`（`Box` 保留，因第 796 行仍在使用）

   修改后：
   ```ts
   import { ShoppingBag, DollarSign, Clock, CheckCircle, Eye, XCircle, Trash2, Box, Bell, BellOff, Search, ClipboardList } from 'lucide-vue-next'
   ```

2. **模板调整**（第 606 行）：`<Box :size="48" />` → `<ClipboardList :size="48" />`

## 假设与决策

1. **假设**：用户"把 div 换成 svg"指将 DashboardView 空状态（div.empty-icon）中的 Box 图标替换为选中的 svg 元素所对应的 ClipboardList 图标。
2. **决策**：DashboardView 第 796 行的 `<Box :size="32" />` 不在用户选中范围内，保持不变，因此 `Box` 保留在 import 中。
3. **决策**：LayoutView 中 `ClipboardList` 和 `Archive` 替换后不再使用，从 import 中清理，避免未使用变量告警。

## 验证步骤

1. 启动开发服务器，查看侧边栏：
   - "桌位管理"导航项图标应为 Armchair（沙发椅图标）
   - "库存管理"导航项图标应为 Package（包裹图标）
2. 进入仪表板页面，触发订单空状态（无订单时）：
   - 空状态图标应为 ClipboardList（剪贴板列表图标），而非 Box
3. 确认仪表板第 796 行处的 Box 图标仍正常显示
4. 运行 `npx vue-tsc -b --force` 确认无类型错误（排除预先存在的 DebugToolsPanel.vue 问题）
