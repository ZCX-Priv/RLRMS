# Checklist

## 401 错误处理
- [x] API 请求返回 401 时自动清除用户状态
- [x] 401 错误时显示会话过期提示
- [x] 401 错误时自动跳转到登录页面
- [x] 跳转时保留原始请求路径作为 redirect 参数

## 会话保活
- [x] 用户登录后启动会话保活定时器
- [x] 定时器每 5 分钟验证一次 token 有效性
- [x] token 验证失败时自动处理登出流程
- [x] 用户手动登出时清除保活定时器

## 状态管理
- [x] auth store 正确记录会话过期时间
- [x] token 即将过期时显示警告提示
- [x] 登出时清除所有认证相关状态

## 用户体验
- [x] 会话过期时显示友好的中文提示
- [x] 登录成功后自动跳转回原请求页面
- [x] 长时间无操作后操作失败有明确提示
